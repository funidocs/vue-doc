# FuniOperatingRecord -操作记录视图组件

#### 参数
| 入参        | 类型   |  默认值  | 说明  |
| --------   | -----:  | :----:  |:----:  |
| api      | string   |   ''     | 接口    |
| requestParmas      | object   |  {}     | 请求参数对象    |


#### vue代码
```html
<funi-operating-record :api='/xx/xx/list' :requestParmas='{id:'1235-4564-5562-54652-1562'}'></funi-operating-record>
```
